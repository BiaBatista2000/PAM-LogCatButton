# App de Avaliação de Filmes 🎬

Esse é um aplicativo Android feito com Jetpack Compose onde o usuário coloca seu nome e o nome de um filme, e depois pode fazer uma avaliação simples usando botões.

## Como funciona

Você digita:
- Seu nome
- Nome de um filme

Depois escolhe uma avaliação:
- I (Insuficiente)
- R (Regular)
- B (Bom)
- MB (Muito bom)

As ações aparecem no Logcat do Android Studio.

## Tecnologias usadas

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio

## Sobre o projeto

Esse app foi feito como atividade de estudo para praticar componentes no Jetpack Compose, uso de estado (remember) e botões personalizados.
